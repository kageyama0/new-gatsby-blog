## Slackって除外検索出来たんですね...

```
-keyword 
```
これで, "keyword"という単語を含まないメッセージだけを検索して取得出来ます。

CIのテスト結果などを、Slackにガンガン飛ばしている時などに、有用かなと思います。

具体的には、あるテストだけすごいコケてるのは分かったから、他のテスト結果だけ見たいよーって時とかですね。
